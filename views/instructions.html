Bruce Analyzer

Hello, and thank you for visiting www.bruceanalyze.me. To use this website first you need to go to the home page. Once you are there, you will see a large textbox. You can use this textbox to search for specific words or phrases in Bruce Springsteen's lyrics. On the left of the textbox there is a dropdown menu with three different modes: "specific", "general", and "song". If you choose "specific", the algorithm will only search for the exact word or phrase you type in. If you choose "general" then the algoritm will search for words with the same root i.e. If you type search for the word run, the algorithm will include words like runs, running, ran, etc. The last mode, "song" will give you the lyrics to a song that you search.
